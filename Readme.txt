
Link to Live Site:
https://ylsnews.netlify.app/

A news aggregation site with Articles and Comments.

You can Comment on Articles and up & down vote them.

You can also up & down vote on Comments.

Articles are visable by topic and sorted by date, votes and comments.

Back End repo:
https://github.com/Siffster85/yls_news

Minimum version of Node: 
v21.5.0

How to run:

git clone https://github.com/Siffster85/nc-news
cd ../yls-news
npm run dev

Have fun.
